# 🏰 El Gran Castillo de Jagua — Website Repository

## 🚀 Overview

This repository contains the official website for **El Gran Castillo de Jagua**, developed by Volta NYC. The site serves as the digital presence for the business, highlighting its cultural significance, offerings, and providing visitors with key information such as location, services, and contact details.

Built using a modern, scalable template, this project ensures consistency across Volta-supported businesses while allowing for full customization of content and branding.

---

## 🌟 About the Business

**El Gran Castillo de Jagua** is a culturally rich establishment inspired by heritage, tradition, and community. This website is designed to reflect the identity of the business while making it easy for customers to explore what it offers and get in touch.

---

## 🛠 Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Hosted on Vercel

---

## 📂 Project Structure

```
src/
  app/
    layout.tsx        → Root layout and global metadata
    page.tsx          → Homepage
    pages/            → Additional pages (About, Contact, etc.)
  lib/
    components/       → Reusable UI components (Navbar, Footer)

public/
  → Static assets (images, favicon, etc.)
```

---

## 🧑‍💻 Development

### Install dependencies

```bash
npm install
npm install --save-dev @types/react
```

### Run locally

```bash
npm run dev
```

The site will be available at `http://localhost:3000`.

## ✏️ Customization Notes
